文章格式指南

1. 支持的文章格式

本网站支持四种主要的文章格式：

  • 简洁版HTML - 推荐格式，支持完整的HTML基本标签、本地图片和本地视频
  • Markdown格式 (.md) - 轻量级标记语言，支持标题、列表、引用、代码块和本地图片
  • TXT格式 - 最简单的格式，支持基本的文本内容和本地图片
  • 完整版HTML - 完全自定义的格式，适合高级用户

2. 简洁版HTML格式（推荐）

简洁版HTML是平衡易用性和功能的最佳选择，特别适合快速创建包含图片和视频的文章。

主要特点：支持本地图片和本地视频直接引用，无需上传到网络服务器！

基本结构

<!DOCTYPE html> <html lang="zh-CN"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>文章标题</title> <meta name="description" content="文章简短描述"> <meta name="date" content="2024-06-24"> <link rel="stylesheet" href="style.css"> </head> <body> <h1>文章标题</h1> <p class="article-date">2024年6月24日</p> <!-- 直接使用本地图片文件名 --> <img src="test_image.svg" alt="图片描述"> <!-- 直接使用本地视频文件名（可选）--> <video controls> <source src="your_video.mp4" type="video/mp4"> 您的浏览器不支持HTML5视频播放。 </video> <p>文章内容段落1...</p> <p>文章内容段落2...</p> <p><a href="index.html">返回首页</a></p> </body> </html>

本地图片使用方法

  • 将图片文件（建议使用SVG格式）放在与HTML文件相同的目录中
  • 在img标签的src属性中直接填写图片文件名，如：src="test_image.svg"
  • 可以在文章的任何位置插入多个图片
  • 系统会自动处理图片的加载和响应式显示

本地视频使用方法

  • 将视频文件（如MP4格式）放在与HTML文件相同的目录中
  • 使用video标签并设置controls属性以显示播放控制
  • 在source标签中，将src属性设置为视频文件名，type设置为正确的MIME类型
  • 支持的视频格式：MP4 (.mp4)、WebM (.webm)、Ogg (.ogg)
  • 可以同时提供多种格式的视频以确保兼容性

视频标签常用属性

  • controls - 显示播放控制
  • width 和 height - 设置视频尺寸
  • autoplay - 自动播放（通常需要静音）
  • loop - 循环播放
  • muted - 静音播放

3. TXT格式

TXT格式是最简单的选择，适合纯文本内容或简单的图文混排。

格式规范

  • 第一行：文章标题
  • 第二行：发布日期（格式：YYYY-MM-DD）
  • 第三行：图片路径（如果有图片，填写图片文件名；无图片则留空）
  • 第四行及以后：文章正文内容
使用本地图片的测试文章 2024-06-21 test_image.svg 这是一篇使用TXT格式创建的测试文章。 TXT格式非常简单，只需要按照固定的格式编写即可。 第一行是文章标题，第二行是日期，第三行是图片路径，第四行开始是正文内容。

4. 文件命名规范

为了确保文章能被正确识别和显示，请遵循以下命名规范：

  • 简洁版HTML文件：wz_数字.html（例如：wz_9.html）
  • TXT文件：wz_数字.txt（例如：wz_8.txt）
  • 图片文件：建议使用SVG格式，文件名可自定义（例如：test_image.svg）
